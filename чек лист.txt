Если есть конкретное числовое указание :
-проверка на минимальное числовое значение
-проверка на максимальное числовое значение
-проверка обработки нулевого значения
Проверка, что при отказе работы функционала отображение корректной ошибки
Если есть требование с датой:
-Високосный день
-Некорректная дата (30 февраля)
-29 февраля в не високосном году
-Различные форматы (Июнь 5, 2001; 60/05/2001; 06/05/01; 05-05-01; 6/5/2001 12:34)
-Перевод на летнее и зимнее время
-Перевод даты вперед-назад
Если есть требование с работой файла:
-Длинное имя файла (> 255 знаков)
-Название с русскими буквами
-Специальные символы в имени (пробел, * ? / \ | < > , . ( ) [ ] { } ; : ' " ! @ # $ % ^ &)
-Не существующий
-Уже существующий
-Нет свободного места на диске
-Доступность хранилища куда грузим файл
-Минимальное место на диске
Проверка действий ввода куда-либо на разных раскладках клавиатуры:
-cut
-copy
-past
